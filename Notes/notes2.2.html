Functions

Parameters

Used to specify a function or result in a function

Example:

func myAge(value:Int) {

let currentAge = 15

print (\(currentAge) is my current age")

}


Arguement Labels

Used to use the function value both in and out of the function

Example:

func friend(name: string) {

print ("This is my friend \(name)")

}

let name = Dave

Return Values

Used when the value of the function needs to be specified

Example:

func multiply(intOne:Int, intTwo:Int) -> Int {

return intOne / intTwo

}

let answer = multiply(intOne: 6, intTwo: 3)

print ("Six divided by 3 is \(answer)"))
