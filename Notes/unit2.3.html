Initializers

Used to return a new instance of a type. For strings it's String.init(), intigers is Int.init(), and for bools it's Bool.init();. The default value, in that order, is "", 0, and false. You can also ommit the "init" and just have, for example, String().


Memberwise Initializers

Used for adding a default value. It also gives a specific value. Such as: let age = Age (count:15)


Custom Initializers

Used to give something a custom value that it calculate/influenced by another. It's used within structs. Pretend the following example is within a struct with 2 values: cm and m :

init(cm : Double) {

self.cm = cm

m =cm * 100

}

^ The value cm is influenced by m. As m increases, so does cm.
