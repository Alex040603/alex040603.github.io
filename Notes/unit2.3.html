Initializers

The general idea of giving default values in a struct. Memberwise and Custom Initializers are specific types.

struct Pencil {

    let color : String : "Blue"

}

let myPencil = Pencil()

print(myPencil.color)


Memberwise Initializers

For giving values for each field in an scruct

struct Shirt {

    let size : String

}

let myShirt = Shirt(size : "Medium")

print(myShirt.size)


Custom Initializers

Used to give something a custom value that it calculate/influenced by another

struct FriendsHeight {

    let heightCm : Int

    init(cm : Double) {

    self.cm = cm

    m =cm * 100

}

^ The value cm is influenced by m. As m increases, so does cm.

It is different to Memberwise Initializers because the values arent set. They can change and be influenced.
