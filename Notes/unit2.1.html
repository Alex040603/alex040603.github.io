.hasSuffix and .hasPrefix

When put behing a let or var value, one that is a string, will identify wheather it has a word before or after. An example would be "Hello Tim". The prefix is "Hello", suffix "Tim"

Eg 3:

Let greetingTim = "Hello, Tim"

if greetingTim.hasSuffix("Tim") {

    print("The suffix is Tim")

}

It will print the text as the suffix is "Tim"


.contains

.contains is used to identify word(s) in a string. For example:

if greeting.contains("Hello") {

    print("We found a greeting")

}

It will print it becuase "Hello everyone" contains Hello
