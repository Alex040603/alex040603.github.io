Functions

Parameters

Used to provide a function with extra information and to give those values an appropraite result

Example:

func myAge(value:Int) {

    let currentAge = 15

    print (\(currentAge) is my current age")

}

print(myAge * 3)


External Argument Labels

Used to use the function value both in and out of the function

Example 1:

func home(country : String, city : String) {

    print ("I live in (\country), (\city)")

}

home(country : "Canada", city = "Vancouver")

Example 2

func home(country : String, in city : String) {

    print ("I live in (\country), (\city)")

}

home(country : "Canada", in : "Vancouver")

Example 3

func home(_country : String, in city : String) {

    print ("I live in (\country), (\city)")

}

home("Canada", in : "Vancouver")


Return Values

Used when the value of the function replaces to function call as a whole

Example:

func divide(intOne:Int, intTwo:Int) -> Int {

    return intOne / intTwo

}

let answer = multiply(intOne: 6, intTwo: 3)

print ("Six divided by 3 is \(answer)"))

Default Parameter Values

Used to declare a default value for a perameter within a function, being able to be called with or without the value. For example, when having a function for calculating someones age, you may set the default value to an integer.
