Outlets and Actions

When you have an object that you want the user to interact with, buttons and interactive elements, you can drag the object to the view controller. When you place the object in the code, you can make an outlet or action. An outlet connects the object to the code, it is usually when an object will change in any way, color, size, etc. The connected prodect should look like:

@IBOutlet weak var labelName: UILabel!


An action is used to run commands when using an object. For example, changing the text on the app when pressing a button is an action. The action should look like:

@IBAction func changeTitle(_ sender: UIButton) {

    mainLabel.text = "this app rocks"

}

In this example, the label is changed to "This app rocks" when the button is pressed.
