Constants,Variable, and Data Types

Constants

  • A value that cannot change
  • Represented as "let"

Variables

  • A value that can change
  • Represented as "var"

Type Inference

Is when the computer guesses, or infers, the value's type in code when it is not declared. For example: -let numberOfSteps = 9- the computer infers that the 9 is an int becuase 9 doesn't have an associated type. Same thing if the number of steps was 8.4, is would be treated as a double. To avoid type inference, it is good to decalre the type with a value as this avoids type inference.


Type annotaion

is when you declare the type of the value in a let or var statement. it is shown as the following: let weight:Double = 45.7 the :Double is declaring that the following value is of that type. Same goes for Int but it is not always nessesary for boolean and strings as type inference will declare it for you.


Types

  • Int
  • Double
  • String
  • Boolean

Int

  • Any natural number (0-infinity)
  • Indicated with "Int"
  • Eg: let numberOfSteps:Int = 600

Double

  • Any decimal number
  • Can assign a whole number as a decimal - Eg: let number:Double = 9
  • The 9 is treated as 9.0 (now a double)

String

  • Words and letters
  • Can be names, made up words, etc (as long as it has letters
  • Is written, and needed, with quotaion marks eg: "a random word"

Boolean

  • A value with only 2 possible outcomes -true or false-
  • Eg: let isMorning:Boolean = True
