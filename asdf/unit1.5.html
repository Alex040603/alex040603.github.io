Shortcuts

Command R is to the the application

Command . is to stop application

Command Enter is to open standard editor. Main screen editor

Command Option + Enter opens the assistant editor. Double circle icon

Command 0 will show you the navigator. Small left side tab

Command Option + 0 opens utility tab. Right side tab

Command Shift + Y opens console. Long bottom tab

Editor Area

Anything opened in the left colunm will be displayed on the Editor Area

Toolbar Area

Hold the different commands such as stop/run the application, open assistant editor, etc

Navigator Area

The "left colunm" mentioned in Editor Area. Lets you choose what file is opened in the Editor Area

Utility Area

Displays the options you can choose from in each folder or application. It holds the details of the projects as well
