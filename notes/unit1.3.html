Operators

Assigning a Value

  • the value of left side of the equal sign is assigned to the one on the right

    • let score = 100

    The 100 is assigned to score

Basic Arithmetics

  • + - * / are used for math
  • Follows the BEDMAS rule

    • 3+3*4 = 15

    Becuase 3*4 is done first

Compound Assignment

    Let score = 0

    score += 3

    This will add 3 to score, which was 0

    score *= 5

    This will multiply 3, your current score with 5 equaling 15

Others

  • For precision, you can assign a type to a math operation

    • let x: Double = 51 let y: Double = 4 let z = x / y

    z has a value of 12.75

Type Conversion

  • You can change the numerical type of a value for a calculation

    • let x = 5

    let y = 3

    let answer = Double(x)/Double(y)

  • The x,y, and answer was changed as a double so that the answer won't round down, thus making the answer more accurate
